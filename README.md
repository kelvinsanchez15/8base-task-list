## About the project

A private task list, featuring all CRUD operations, a cloud function web hook and a custom Material UI theme.

## Features

- [x] Full CRUD operations
- [x] Custom Cloud function
- [x] Amazon Cognito sign-in service
- [x] Custom MUI theme
